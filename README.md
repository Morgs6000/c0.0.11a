## Adições
### Jogabilidade
Controles
- Clique com o botão direito do mouse alterna entre os modos de colocação e destruição.
  - A seleção de um bloco agora mostra uma sobreposição branca em todos os lados do blocos. 
- Esc desconecta o mouse.
